🏢 Human Resource Management System (HRMS)

A modern Human Resource Management System (HRMS) that simplifies and automates essential HR operations. The application provides a secure, centralized platform for managing employees, attendance, leave requests, and payroll while supporting role-based access for both employees and administrators.

✨ Features

- 🔐 Secure User Authentication (Sign Up / Sign In)
- 👥 Role-Based Access Control (Admin & Employee)
- 👤 Employee Profile Management
- 🕒 Attendance Tracking (Check-in / Check-out)
- 📅 Leave & Time-Off Management
- 💰 Payroll Information
- ✅ Leave Approval Workflow
- 📊 Interactive Dashboards
- 📱 Responsive User Interface

🎯 Project Objectives

- Digitize and streamline HR processes
- Reduce manual paperwork and administrative effort
- Improve employee self-service capabilities
- Ensure secure access through authentication and authorization
- Provide a centralized employee management platform

👤 User Roles

Admin / HR Officer

- Manage employee records
- Monitor attendance
- Approve or reject leave requests
- Manage payroll information
- View reports and analytics

Employee

- View and update profile
- Mark attendance
- Apply for leave
- Track leave status
- View payroll details
